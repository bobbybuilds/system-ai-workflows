# Setup Checklist

## Folder Setup
- create project folder in /projects
- create core subfolders
- add README files where needed

## Git Setup
- run git init
- add files
- commit initial structure

## GitHub Setup
- create repo without README
- add remote origin (SSH)
- push initial commit

## Verification
- confirm files appear on GitHub
- confirm no auth prompts (SSH working)
- confirm structure is correct
