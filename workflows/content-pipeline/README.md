# Content Pipeline

This workflow defines the standard process for turning one core topic into a full content stack.

## Goal

Create one source asset, then repurpose it into multiple formats without rebuilding from scratch every time.

## Pipeline Flow

1. Choose topic
2. Create core long-form content
3. Extract key points
4. Repurpose into short-form assets
5. Publish and distribute
6. Track outputs and improve next cycle

## Standard Outputs

- blog post
- short-form video script
- social captions
- carousel/post copy
- Pinterest pin copy
- YouTube short support copy

## Principle

One idea should produce multiple assets.
