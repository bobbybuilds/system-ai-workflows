# Workflow Map

## Stage 1 — Topic Selection
Choose a topic based on:
- audience need
- search value
- clarity potential
- monetization relevance
- repurposing potential

## Stage 2 — Core Content Build
Create the main long-form piece first.

## Stage 3 — Message Extraction
Pull out:
- core claim
- strongest hook
- biggest insight
- key takeaway

## Stage 4 — Repurposing
Turn the core piece into:
- short-form videos
- carousels
- Pinterest assets
- post captions

## Stage 5 — Publishing
Publish in the correct order:
1. core content
2. short-form content
3. discovery assets

## Stage 6 — Review
Evaluate clarity, usefulness, and performance.
