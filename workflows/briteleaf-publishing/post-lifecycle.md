# Post Lifecycle

## Phase 1 — Idea
Start with a topic worth breaking down.

## Phase 2 — Core Build
Create the main article as the source asset.

## Phase 3 — Extraction
Pull hooks, key points, risks, takeaways, and CTAs.

## Phase 4 — Repurposing
Adapt the source asset into platform-specific outputs.

## Phase 5 — Publishing
Publish the article first, then use support content to drive attention back to it.

## Phase 6 — Review and Reuse
Review what worked, improve the process, and reuse the topic angle where appropriate.
